# agentic-skills

Claude Code 技能库 / Claude Code Skills Collection

把开发成熟的 Claude Code 技能统一入库，脱敏后公开分享。

## Skills index

| 技能 | 功能描述 | 状态 |
|------|----------|------|
| [双语阅读论文](skills/双语阅读论文) <br>(bilingual-paper-reader) | 把论文 PDF 生成自包含本地双语速读网页：左侧 PDF 原文、右侧中文极简讲解+英文对照，顶部 toggle 一键显隐英文，Chrome 双击即用无需联网 | ✅ ready |

## 安装使用

每个技能目录下都有 `README-配置指南.md`，按其说明放置文件到 `~/.claude/commands/` 即可。
